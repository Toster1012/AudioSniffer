# AudioSniffer

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

AudioSniffer - инструмент для захвата и анализа аудиопотоков. Проект использует Python-бэкенд и .NET-компоненты, взаимодействующие через локальный веб-сервер. Анализ аудио файла происходит по средвом логики, а не нейронных сетей.

---

## Требования

Перед запуском убедитесь, что на вашей машине установлено следующее:

|   Компонент  | Минимальная версия |
|--------------|--------------------|
| Python       |        3.10+       |
| .NET SDK     |         10         | 
| .NET Runtime |         9.0        |

---

## Запуск

### 1. Первый запуск (установка зависимостей)

1. Запустите **`start_server.bat`** - откроются два консольных окна.
2. Дождитесь окончания установки зависимостей в обоих окнах.
3. Нажмите **Enter** в каждом окне и **закройте обе консоли**.

### 2. Повторный запуск

Снова запустите **`start_server.bat`**.

> Первый запуск необходим только один раз для установки всех зависимостей.

### 3. Открыть интерфейс

После запуска сервера откройте браузер и перейдите по адресу:

```
https://localhost:8000
```

> Браузер может предупредить о самоподписанном сертификате. Нажмите "сделать исключение для сайта".

---

## Возможные проблемы

- **Порт 8000 занят** - завершите процесс, занимающий порт, или измените конфигурацию.
- **Python не найден** - убедитесь, что Python 3.10+ добавлен в переменную среды `PATH`.
- **.NET SDK не найден** - установите .NET SDK 10 с [официального сайта](https://dotnet.microsoft.com/download).
